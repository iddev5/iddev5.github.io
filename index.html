IDDEV'S Homepage

Hello, I am Ayush, also known as idhank or iddev. I am a game developer, designer and programmer. I am a Linux user and enthusiast. I also do a little bit of web development. I like Open Source softwares and I support FOSS.

My programming languages are C, C++(Orthodox), Lua, but I also know some NodeJS, HTML & CSS(this page). Game engines/frameworks I use are Deccan Engine(my own project), olcPixelGameEngine, Love2D, GDevelop. My Linux distribution is Linux Mint with Xfce. My favourite web framework is Svelte. I use Vim text editor.

Projects:

  • Deccan Engine is an Open-Source Cross-platform 2D Game Engine written in C11 and is powered by SDL. [Website][GitHub][Trello]

  • INTP is an interpreted programming language written in C and was meant to be used as an embedded language, though it never advanced to that stage and currently barebones and abandoned. [GitHub]

  • Raven2D Engine is a C++ Object-Oriented game engine based on SDL2. My first successful attempt at making a game engine. [GitHub]

  • Rockett is a casual, arcade-style game about controlling Rockets with wobbly physics, to shoot down enemy planes while protecting friendly forces. [Itch.io]

  • Overwatcher Bot is an auto-moderation bot written in NodeJS as my first real(or sensible) NodeJS practice project. Not actively developed or hosted, you are free to fork and learn from it! [GitHub]
