Some common embeddable virtual machines are:

  • Java Virtual Machine (JVM)
  • Web Assembly (WASM)
  • BEAM VM
  • .NET CLR (and Mono) - The .NET Common Language Runtime is a virtual machine part of .NET Framework. It can optionally convert the IR code into machine code Just-In-Time (JIT) for faster execution. As always, it provides memory management, type safety, expection handling, garbage collection etc. The large ecosystem of .NET comprising of numerous frameworks and tools makes it an enticing choice.
  • ART (and previously Dalvik) - Dalvik was the original virtual machine used in Android. It was responsible for running all programs written for Android. Dalvik bytecode format is still used as a distribution format but Dalvik itself is not longer available in newer versions of Android.
  • LLVM

Just for info

Python also has an internal bytecode but its not standard and not stable. It may change across different versions of python and it designed specifically for python.

Parrot VM is a discontinued register-based VM designed to run dynamic languages. It was designed by Perl community for Raku and was later abandoned in favour of MoarVM.
