Just enough Git for a hackathon!

This is a short tutorial to understand and use enough of Git you need for a hackathon

What is Git?

Git is a (collaborative) decentralized version control system. Let's understand each part one by one.

  • Version control system: A version control system is a software which is used to store your code in such a way that each set of changes is stored separately. For example, you have a website initially. You add a home page to it. Make it one part. Now you added another page - about, this will be stored in a different part. We will learn why we need it soon!
  • Collaborative: Git is mostly a collaboration tool, it lets multiple people to work on a project at the same time.
  • Decentalized: The code is not stored at one place. It is present in the devices of all collaborators working on the project. This feature also makes it a bit complicated to work with, especially in a non-networked ecosystem, that's where GitHub comes in.
